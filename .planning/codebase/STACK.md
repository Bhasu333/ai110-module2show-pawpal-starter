# Tech Stack

This document lists the technologies, libraries, and frameworks used in the PawPal+ project.

## Core Language & Runtime
- **Python**: 3.x (runs locally)
- **Streamlit**: Used to build the interactive web UI (`app.py`)
- **pytest**: Used for unit testing the logic layer (`tests/test_pawpal.py`)

## Dependencies
Defined in `requirements.txt`:
- `streamlit`
- `pytest`

## Configuration
- No external configuration files required. Local execution via virtual environment.
